Question 2: how to set up VSCode

Visual Studio Code (VS Code) is a source-code editor that can be installed on any operating system (Windows, Linux, MacOS) in fact you can choose the appropriate operting system when istalling the software (check in Q1). This code editor can also interact with different programming languages (Python,C, C#, etc) for this reason you need to apply a couple of extensions that will assist you in coding.

Studio Code comes equipt with most of the things you need, so setup is easy. here are some of the changes you can make to personalise VS Code.

To choose the background theme of your code editor
1. Go to the menu bar - the top pane on the page (this is where you find file, edit, selection etc).
2. click on file and scroll down until you find preferences.
3. Hover over preferences to find more option including Theme or for a shortcut use ctrl + K and ctrl + T
4. Once there choose the them use prefer light or dark


similarlly if you prefer to change the font family do the following:
1. Go to the menu bar - the top pane on the page (this is where you find file, edit, selection etc).
2. click on file and scroll down until you find preferences.
3. Hover over preferences to find more option including settings alternatively use the shortcut ctrl + ,
4. choose whether to leave it at the default font family or change to suit your desire
