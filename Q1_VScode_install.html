Question 1: Installing VS Code on windows 11

1. on your browser (Chrome, windows explorer, etc) search the VS Code site for downloads https://code.visualstudio.com/download
2. Choose the appropriate option for your local machine (Windows, MacOS, Ubuntu).
3. Once download is complete - Launch the software (either right click and choose to open or double click).
4. Agree to the terms and conditions

5. click next
6. Choose the location for VS code to install - a default location will pre-populate however if you can install it in the location of your choice

7. Software now ready to install so click install.
8. All the software to install - it may take a few minutes
9. Once installtion is complete all you need to do is click finish and it will launch VSCode
